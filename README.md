# 最近访问标签页（回到刚才）

从 Claude Code（或任何应用）切回 Chrome 时，一下子找不到刚才在看的页面？
在任意网页上**点右键 → 「最近访问的标签页」**，直接列出最近访问的 5 个标签（含"几分钟前"），点击跳转，跨窗口有效。

## 安装（加载未打包扩展）

1. Chrome 地址栏打开 `chrome://extensions`
2. 右上角打开「开发者模式」
3. 点「加载已解压的扩展程序」，选择本文件夹 `recent-tabs-menu/`

## 使用

- **主入口**：任意网页右键 → 「最近访问的标签页」→ 子菜单点击跳转
- **兜底入口**：点工具栏扩展图标，弹窗列出最近 10 个标签，点击跳转
  （`chrome://` 设置页、Chrome 商店页面右键菜单不可用，用这个入口）

## 说明与限制

- 数据来源是 Chrome 每个标签页自带的 `lastAccessed`（最后访问时间），无额外追踪、无网络请求、不存储任何数据
- 需要 Chrome 121 或更高版本
- 列表排除当前正在看的标签页，显示的是"上一个及更早"
- 菜单里的"X 分钟前"在标签切换时实时刷新，静止状态下每 5 分钟刷新一次
